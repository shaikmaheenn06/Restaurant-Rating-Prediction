# Restaurant Rating Prediction

## Objective
Build a machine learning model to predict restaurant ratings using regression algorithms.

## Algorithms Used
- Linear Regression
- Decision Tree Regression

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## Result
Decision Tree Regression achieved better performance than Linear Regression, with a lower MSE and a higher R² score.
